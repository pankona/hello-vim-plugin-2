# Makasero プロジェクト概要

## プロジェクトの目的

Makaseroは、OpenAI GPT-4を活用したコード分析・改善支援のためのCLIツールです。主な目的は：

1. コードの説明生成
2. インタラクティブなコードレビュー
3. コード改善提案の自動生成と適用

## 主要機能

### 1. コマンドライン機能
- `explain`: コードの説明生成
- `chat`: AIとのインタラクティブな対話
- JSON形式での入出力サポート

### 2. コード提案システム
- AIによる改善提案の生成
- 差分の自動検出と適用
- ユーザー承認フローの統合

### 3. 安全性機能
- バックアップ作成
- 変更の検証
- ロールバック機能
