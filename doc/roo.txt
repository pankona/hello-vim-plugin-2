1. 概要                                                    *roo-introduction*

Rooは、AI支援によるコーディング支援を提供するVimプラグインです。
Roo Code VSCode拡張の機能をVimで実現することを目指しています。

主な機能：
- AIによるコード生成と編集
- モードベースのアクセス制御
- ファイル操作の自動化
- ブラウザ操作の統合
- MCPサーバーとの連携

==============================================================================
2. 要件                                                    *roo-requirements*

- Vim 8.1以上
- Python 3.6以上
- curl
- (オプション) Node.js 14以上 (MCPサーバー機能用)

==============================================================================
3. インストール                                            *roo-installation*

vim-plugを使用する場合：
>
    Plug 'username/hello-vim-plugin-2'
<

手動インストール：
>
    git clone https://github.com/username/hello-vim-plugin-2.git ~/.vim/pack/plugins/start/hello-vim-plugin-2
<

==============================================================================
4. 使用方法                                                *roo-usage*

基本的な使用方法：

1. タスクの開始:
>
    :RooTask "Create a new function to calculate fibonacci numbers"
<

2. モードの切り替え:
>
    :RooMode           " 現在のモードを表示
    :RooModeSwitch code " コードモードに切り替え
<

==============================================================================
5. コマンド                                                *roo-commands*

                                                            *:RooTask*
:RooTask {task}
    新しいタスクを開始します。

                                                            *:RooMode*
:RooMode
    現在のモードを表示します。

                                                            *:RooModeSwitch*
:RooModeSwitch {mode}
    指定したモードに切り替えます。

==============================================================================
6. 設定                                                    *roo-configuration*

プラグインの設定例：
>
    " API設定
    let g:roo_api_key = 'your-api-key'
    let g:roo_model = 'gpt-4'

    " デフォルトモード
    let g:roo_default_mode = 'code'

    " 履歴ファイルの場所
    let g:roo_history_file = '~/.vim/roo_history.json'
<

==============================================================================
7. モード                                                  *roo-modes*

プラグインは以下のモードを提供します：

Code モード                                                *roo-mode-code*
    プログラミングと実装に特化したモード。
    すべてのツールにアクセス可能。

Architect モード                                          *roo-mode-architect*
    アーキテクチャ分析とドキュメンテーション用のモード。
    Markdownファイルの編集のみ許可。

Ask モード                                                *roo-mode-ask*
    質問応答と説明用のモード。
    読み取り操作とMarkdownファイルの編集のみ許可。

==============================================================================
8. トラブルシューティング                                  *roo-troubleshooting*

よくある問題と解決方法：

Q: APIキーが認識されない
A: .vimrcでの設定を確認してください。

Q: モード切り替えができない
A: プラグインが正しくロードされているか確認してください。

Q: ファイル操作でエラーが発生する
A: ファイルのパーミッションとモードの制限を確認してください。
